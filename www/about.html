Features Addressed By My Web Server

Static File Serving

Serves static files from the specified web root directory ("./www/"), supporting various file types including HTML, CSS, JavaScript, images, audio, and video.

MIME Type Detection

Automatically detects and sets appropriate MIME types for different file extensions, ensuring proper content delivery.

Multi-threaded Architecture

Utilizing a thread pool with 5 worker threads to handle multiple connections concurrently. Each client connection is handled in a separate thread for optimal performance.

HTTP Protocol Support

Implements basic HTTP/1.1 protocol with comprehensive GET request handling and appropriate HTTP headers including status codes, content types, and content lengths.

Error Handling

Provides custom error pages for common HTTP errors (400, 404, 405) with fallback to default error messages when custom pages are unavailable.

Logging and Debugging

Comprehensive logging system that prints received HTTP requests for efficient debugging and monitoring.

Connection Management

Efficient connection handling with automatic closure after each request using the Connection: close header.

Default Page Handling

Automatically serves "index.html" when the root path ("/") is requested, providing seamless default page functionality.
